    激光照明

    激光原理

    目前激光方式基本靠受激辐射产生。光子射入物质诱发电子从高能阶跃迁到低能阶，并释放光子。

    激光特点

    1. 方向性好
    2. 单色性好
    3. 能量集中
    4. 相干性好

    激光分类

    按能量分：低、中、高能量 按波动分：可见光、红外、紫外、X光等 按激光器分：为固体、气体、液体、半导体和染料等几种类型

    激光组成

    激光发生装置（气体激光管、半导体激光器等）+聚焦组件（包括扩束、聚焦、调节）+激光电源（供电、滤波、稳压、调节）

    激光应用

    1. 工业应用：主要是测量、加工等
    2. 医疗用途：各类微创手术
    3. 军事用途：测距，制导，通信，武器
    4. 日常应用：打印机、扫描器等，霓虹灯等

    激光Laser发展

    1. 1917年爱因斯坦《关于辐射的量子力学》预言了受激辐射发光的可能性，即激光的存在
    2. 20世纪50年度，汤斯和肖洛的光激射器理论；激光器方案的提出
    3. 1960年梅曼制成世界上第一台激光器
    4. 目前各类激光器不断的发展

    激光照明介绍

    激光照明目前采用半导体激光器为主，然后激发单色光

    半导体激光器介绍

    半导体二极管激光器是以一定的半导体材料做工作物质而产生受激发射作用的器件。其工作原理是,通过一定的激励方式,在半导体物质的能带(导带与价带)之间，或者半导体物质的能带与杂质(受主或施主)能级之间，实现非平衡载流子的粒子数反转，当处于粒子数反转状态的大量电子与空穴复合时，便产生受激发射作用。 半导体激光器的激励方式主要有三种：即电注入式，光泵式和高能电子束激励式。

    激光照明存在的优势

    效率高，由于激光的相位频率相同，出来的相干光，能量聚大

    激光照明白光方式：

    1. 蓝光激光激发荧光粉实现白光照明
    2. 红绿蓝激光合成白色激光或真彩色光照明

    激光照明目前需要解决问题

    1. 激光器问题：主要集中在成本，工艺，稳定性，大功率等问题
    2. 散射问题：激光是相干光源，而我们照明大部分需要是非相干光源。
    3. 散热问题：由于出光后能量大，芯片端散热需要解决
    4. 白光合成：目前方案工艺成本上都欠缺。

    目前激光照明的发展

    目前激光照明的发展进展主要集中在上游厂家，即激光器的研发及白光和成。用于照明的激光器集中在半导体的激光器，目前的半导体激光器比较成熟的方案用于条形扫码、激光打印等领域。照明领域的激光器目前还处于不断改进阶段。

    主要激光器研发厂家

    日亚，罗芬，欧司朗，炬光科技，Lumentum， 夏普等

    激光照明目前处于阶段

    目前类似LED照明发展历史中的直插单色LED阶段。一旦大功率激光器及白光方案研发成功，真正进入了激光照明。

    主要激光照明产品

    激光车灯，激光手电筒，激光景观照射，激光投影，激光电视
